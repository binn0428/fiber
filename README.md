# 光纖管理網頁系統 (Fiber Management System)

這是一個基於 Vite + Vanilla JS + Supabase 的光纖管理系統。

## 功能特色
- **儀表板**: 檢視全網與各站點的 Port 使用統計。
- **光纖架構圖**: 視覺化站點分佈，點擊站點可查看詳細 Port 列表。
- **資料管理**: 檢視、搜尋與編輯線路資訊。
- **Excel 整合**: 支援多頁籤 Excel 匯入 (自動解析站點與線路) 與資料匯出。
- **Supabase 整合**: 資料同步至雲端資料庫。

## 快速開始

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```
開啟瀏覽器訪問顯示的 URL (通常是 http://localhost:5173)。

### 3. Supabase 設定
首次使用時，請至系統的「上傳 / 匯出」頁面，輸入您的 Supabase URL 與 Anon Key。

**Supabase Table Schema (資料表結構)**
請在您的 Supabase SQL Editor 執行以下指令以建立必要的資料表：

```sql
create table ports (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  site_name text, -- 站點名稱 (對應 Excel 分頁名稱)
  line_name text, -- 線路名稱
  port_number text, -- Port 編號
  usage text, -- 用途
  remarks text -- 備註
);
```

### 4. Excel 格式說明
匯入的 Excel 檔案應包含多個分頁 (Sheets)，每個分頁代表一個「站點」。
第一列為標題列，建議包含以下欄位 (系統會自動模糊比對)：
- `線路名稱` (或 `Line Name`)
- `Port`
- `用途` (或 `Usage`)
- `備註` (或 `Remarks`)

若無標題，系統將預設採用第 1, 2, 3, 4 欄分別為 線路, Port, 用途, 備註。

## 開發技術
- Frontend: HTML5, CSS3, JavaScript (ES Modules)
- Build Tool: Vite
- Database: Supabase
- Excel Lib: SheetJS (xlsx)
